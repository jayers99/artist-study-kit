# Research prompt — A formal framework for analyzing an artwork's visual style for studio practice

I'm building a tool that analyzes images of artworks and produces **practical notes to help an artist prepare a master study**. I need a rigorous, reusable **taxonomy and method for formal/visual analysis** — not biography or interpretation, but how the image *works* visually.

Research and synthesize established frameworks into a single structured system, drawing on: the **elements of art** (line, shape, form, value, color, texture, space) and **principles of design** (balance, contrast, emphasis/focal point, rhythm, movement, proportion, unity, pattern, hierarchy); **art-historical formal analysis** method; **color theory** (palette, harmony, temperature); **value/notan** and light-and-shadow structure; **composition** (geometry, dynamic symmetry, negative space); and **mark-making / brushwork / surface handling**.

Produce:
1. **A layered taxonomy** of analysis dimensions, organized from big-picture (composition, value structure, color key) down to detail (edge control, brush economy, surface texture).
2. **For each dimension**, what to look for and the vocabulary to describe it precisely.
3. **A repeatable analysis checklist/workflow** an LLM could apply to a single painting image to generate notes answering: What is the image doing visually? What should I notice first? What are the dominant design decisions? Which techniques should I imitate? What traps/misconceptions should I avoid?
4. **Study-exercise suggestions** tied to dimensions (e.g. value thumbnails, master-copy passages, palette-limiting drills) that help internalize a style.

Favor sources from art education, atelier/academic training, and formal-analysis guides (museums, universities, respected instructors). Conclude with a compact, reusable **analysis template** I can apply per selected work.
